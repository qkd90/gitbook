1.首先思路就是倒序排列，用limit限制输出。
2.但是如果不存在则不会返回null，因此需要作为临时表

```sql
# Write your MySQL query statement below
SELECT
    (SELECT DISTINCT Salary
        FROM Employee
        ORDER BY Salary DESC
        LIMIT 1,1) AS SecondHighestSalary
```
