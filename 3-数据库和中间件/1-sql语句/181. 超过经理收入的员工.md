```sql
select a.Name as 'Employee'
from Employee as a,Employee as b
where a.ManagerId=b.Id
        and a.Salary>b.Salary;

```
1.employee的表出现了两次，分别命名为a，b，因为只有一个表要使用两次，为了避免二义性才用不同的名字
