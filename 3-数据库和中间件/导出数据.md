## 用户id-excel处理

### 1.直接用文本编辑器（Notepad++ / VS Code）

1. 粘贴原始 ID
2. **替换换行符**
   - 查找：`\n`
   - 替换为：`','`
3. 在最前面加 `'`
4. 在最后加 `'`

### 2.Excel / WPS（最常用）

假设 ID 在 **A1:A4**

1️⃣ 在 B1 输入：

```
="'"&A1&"'"
```

向下拖拽填充

👉 **Excel 365 / WPS 新版**可以直接用： **有长度限制**

```sql
=TEXTJOIN(",",TRUE,"'"&A2:A4&"'")
```
