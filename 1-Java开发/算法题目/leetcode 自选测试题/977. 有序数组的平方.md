一、这题太简单就是直接平方，然后排序

```python
class Solution(object):
    def sortedSquares(self, A):
        """
        :type A: List[int]
        :rtype: List[int]
        """
        for i in range(len(A)):
            A[i] = A[i] **2
        A.sort()
        return A
```
